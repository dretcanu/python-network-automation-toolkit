# Lessons Learned

## What I Learned

This project strengthened my understanding of Python automation for networking and infrastructure tasks.

Key learning points:

- How to read network data from text and CSV files
- How to scan IP subnets using Python
- How to use `ipaddress` for subnet handling
- How to use `subprocess` to run system ping commands
- How to generate structured CSV and TXT reports
- How to organise a Python project for GitHub
- How to use a virtual environment
- How to build a simple CLI menu system

---

## Limitations

The current configuration backup module is a simulator and does not yet connect to real network devices.

The network health check depends on ICMP being allowed and devices being reachable from the current machine.
